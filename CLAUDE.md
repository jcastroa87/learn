# Project: Learn

## Stack

- Backend: Laravel 12 PHP 8.3+
- BD: Postgres
- Frontend: TypeScript 5.x strict Next.js 15

## Code Style

PHP 8.3+ (backend), TypeScript 5.x strict (frontend): Follow standard conventions

## Frontend design style

- Reference: https://keiki.app/
- Clean, modern
- Good typography hierarchy
- Proper whitespace and spacing
- Subtle shadows and borders
- Coherent color palette (prefer neutral base + one accent color)
- Smooth hover states and transitions
- Mobile responsive by default
- Tailwind CSS via CDN for utility classes
- No jQuery, vanilla JS only
